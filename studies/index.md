# Case Studies

Real-world examples of applying the AILIS framework to understand and analyze AI systems.

## Purpose

Case studies demonstrate how AILIS can be used to:

- Analyze existing AI systems and map them to layers
- Identify architectural patterns and anti-patterns
- Compare different system designs
- Guide architectural decisions

## Contributing Case Studies

We're especially interested in case studies that:

- **Analyze real systems** - Map actual AI applications to AILIS layers
- **Compare architectures** - Show different approaches to similar problems
- **Highlight challenges** - Document where the model works well and where it struggles
- **Propose improvements** - Suggest refinements based on real-world usage

## Format

Case studies should include:

1. **System Overview** - What system are you analyzing?
2. **Layer Mapping** - How does it map to AILIS layers?
3. **Key Insights** - What did you learn?
4. **Challenges** - Where did the model fall short?
5. **Recommendations** - How could AILIS be improved?

See our [Contributing Guide](../CONTRIBUTING.md) for submission guidelines.

## Examples We'd Love to See
